# 🚗 Luxury Car Rental System

## 📌 Project Overview
The Luxury Car Rental System is a full-stack web application developed as part of a university project. It allows users to browse and rent luxury cars online with a simple and user-friendly interface.

The system also includes an admin panel for managing cars, users, and booking records.

---

## 🚀 Key Features

### 👤 User Features
- User registration and login system
- Browse available luxury cars
- Add cars to cart
- Secure checkout and booking process
- View booking history

### 🔧 Admin Features
- Admin dashboard
- Add, update, and delete cars
- Manage users
- Monitor rental records and transactions

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP (Apache)
